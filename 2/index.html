Why learn angular

If you're using Javascript, Angular is good for creating dynamic websites.

  • It helps you organize your JavaScript
  • It helps create responsive (as in fast) websites
  • It plays well with jQuery
  • It's easy to test

"A client-side JS framework for adding interactivity to HTML."

You can add interactivity through directives. You use directives by adding tags onto HTML elements. Directives: it's how we bind the behavior.
